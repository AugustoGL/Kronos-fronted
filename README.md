# Kronos — Dashboard

Dashboard de **gestión de horarios escolares**, desarrollado con React, TypeScript, Vite y [Mantine](https://mantine.dev/).

Permite administrar de forma centralizada:

- **Horarios** de cursada
- **Materias** y su carga horaria
- **Profesores** y su asignación a materias/horarios
  
Este repositorio contiene el **dashboard** (frontend) del proyecto.

## Stack

- React + TypeScript
- Vite
- Mantine (UI)

## Estado

Proyecto activo, en desarrollo continuo.
